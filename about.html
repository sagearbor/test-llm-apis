About DIAL

DIAL (DCRI Intelligent AI Liaison) is a secure web interface for testing and interacting with multiple Azure OpenAI language models.

üéØ Key Features

Multiple LLM Models

Choose from several pre-configured Azure OpenAI models optimized for different tasks:

  • Coding LLM - Specialized for code generation and programming tasks
  • Smallest LLM - Lightweight model for quick, simple queries
  • All-Around LLM - General purpose model for most tasks
  • Best LLM - Advanced reasoning for complex problems
  • Longest LLM - Extended context window for large documents

Advanced Parameter Control

Customize model behavior to suit your needs:

  • Token Limits - Control response length (1,000 - 50,000 tokens)
  • Reasoning Effort - Adjust depth of analysis (for reasoning models)
  • Verbosity - Control detail level in responses

File Upload Support

Attach documents to your queries for analysis and discussion. Supported formats include text files, PDFs, images, and more.

üîí Security & Privacy

‚úì Azure Secure

Your data is protected through multiple layers of security:

  • Azure OpenAI Integration - All queries are processed through Microsoft Azure's secure cloud infrastructure
  • Enterprise-Grade Security - Azure OpenAI complies with SOC 2, HIPAA, ISO 27001, and other industry standards
  • Data Residency - Your data stays within Azure's secure data centers
  • No Third-Party Sharing - Conversations are not used to train models or shared with other parties
  • Encrypted Transmission - All data is encrypted in transit using industry-standard protocols
